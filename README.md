## Sobre o Projeto

Aplicação desenvolvida em React + TypeScript seguindo o layout proposto no Figma.
A página exibe uma vitrine de produtos consumindo dados via JSON e permite a interação com modal ao clicar em um produto.

Além do escopo obrigatório, foram aplicadas melhorias de SEO técnico, acessibilidade e performance como diferencial.

### Tecnologias Utilizadas
- React
- TypeScript
- Vite
- Sass (SCSS)
- ESLint
- HTML5 Semântico

### Requisitos Atendidos
- Desenvolvimento em React + TypeScript
- Consumo de produtos via JSON
- Modal com informações do produto ao clicar
- Layout fiel ao Figma (pixel a pixel)
- Utilização de pré-processador Sass
- Sem uso de bibliotecas UI (Bootstrap, etc.)
- Componentização organizada

### Diferenciais Implementados (Pontos Extras)
- Boas práticas de SEO técnico e on-page
      - Meta tags (title, description, Open Graph, Twitter)
      - Canonical
      - robots.txt e sitemap.xml
- HTML semântico (main, section, hierarquia correta de headings)
- Acessibilidade (aria-label, alt descritivo, classe sr-only)
- Otimização de imagens (loading="lazy" e decoding="async")
- Responsividade adicional (Mobile a partir de 320px)

## Como Rodar o Projeto
```bash
git clone <url-do-repositorio>
```
Instale as dependências
```bash
npm install / npm i
```

Execute o projeto
```bash
npm run dev
```

O projeto estará disponível em:
```bash
[npm run dev](http://localhost:5173)
```

## Build para Produção
```bash
npm run build
```
